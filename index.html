Block Elements

<address>

The <address> element indicates that the enclosed HTML provides contact information for a person or people, or for an organization.

html code

<address>
Jerry Smith<br>
Washington,<br>
USA,<br>
Earth
</address>

browser example

Jerry Smith
Washington,
USA,
Earth

<article>

The <article> element represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable (e.g., in syndication).

html code

<article>
<h2>Jerry Smith</h2>
<p>Jerry is the husband of Beth Smith, the father of Summer Smith and Morty Smith, and the son-in-law of Rick Sanchez and Diane Sanchez</p>
</article>

browser example

Jerry Smith

Jerry is the husband of Beth Smith, the father of Summer Smith and Morty Smith, and the son-in-law of Rick Sanchez and Diane Sanchez

<aside>

The <aside> element represents a portion of a document whose content is only indirectly related to the document's main content.

html code

<aside>
<h3>Gas Giant</h3>
<p>Jupiter is twice as large as all the other planets combined.</p>
</aside>

browser example

Gas Giant

Jupiter is twice as large as all the other planets combined.

<blockquote>

The <blockquote> element indicates that the enclosed text is an extended quotation. Usually, this is rendered visually by indentation . A URL for the source of the quotation may be given using the cite attribute, while a text representation of the source can be given using the cite element.

html code

<blockquote cite="https://rickandmorty.fandom.com/wiki/Jerry_Smith">
Jerry has scruffy, dirty brown hair, with slight curls. He has dimples and wears a dark green shirt, with a brown and beige stripe in the middle. He also wears light blue long pants and black shoes.
</blockquote>

browser example

Jerry has scruffy, dirty brown hair, with slight curls. He has dimples and wears a dark green shirt, with a brown and beige stripe in the middle. He also wears light blue long pants and black shoes.

<h1> - <h6>

The <h1> - <h6> elements represent six levels of section headings. <h1> is the highest section level and <h6> is the lowest.

html code

<h1>This is heading 1</h1>

<h2>This is heading 2</h2>

<h3>This is heading 3</h3>

<h4>This is heading 4</h4>

<h5>This is heading 5</h5>

<h6>This is heading 6</h6>

browser example

This is heading 1

This is heading 2
